# Trabajo Final Mariposas

Sitio web educativo sobre mariposas — Trabajo Práctico Final de Informática General.

## Cómo ver el sitio

Abrir `index.html` en cualquier navegador (Chrome, Edge o Firefox). No requiere instalación:
es un sitio estático hecho con HTML, CSS y JavaScript.

## Tecnologías

- HTML5 semántico
- CSS3 propio (`assets/css/custom.css`) + Bootstrap 5.3.3 (grilla, navbar, cards, formularios, modal, accordion)
- JavaScript nativo (`assets/js/`): validación de formulario, galería interactiva y juego en canvas

## Estructura

16 páginas HTML (home, 6 secciones principales, 6 subpáginas, 2 páginas complementarias y el
juego), organizadas junto a `assets/css`, `assets/js` y `assets/img`. Más detalle en
[registro-desarrollo.md](registro-desarrollo.md).
